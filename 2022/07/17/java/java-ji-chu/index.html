Java 基础知识

数据类型

Java中所有的数值类型字节数与平台无关

1字节 2字节 3字节 4字节
byte short int(默认整型) long(后缀L/l)
boolean char float(后缀F/f) double(默认浮点,后缀D/d可省略)
  • Java没有无符号形式的整型，非要使用需将整型解释为无符号数
    如：Byte.toUnsignedInt(aByte)
  • float有效位数：6~7, double有效位数: 15
  • 三个特殊的浮点值
    Double.POSITIVE_INFINITY: 正无穷
    Double.NEGATIVE_INFINITY: 负无穷
    Double.NaN: 非数
    • 所有非数都不相等
    • Double.isNaN(var) 判断非数
  • 没有空字符，‘’ 是错误的；有空字符串""

枚举类型

enum Name {
    value1,
    value2,
    value3
};
Name var = Name.value; 
Name var = null;  
```  


注：枚举是种特殊的类，不能在`方法`中定义  
    可作为类的`属性`，也可在`类外`

## 常变量  

### 变量

- 声明  
    `类型 变量名;`  
- 初始化  
    `变量名 = 值;`  
- `Java10` 开始，  使用`var`  
定义`局部变量`(不是类的属性) 可以从初始值推断其类型而不需要指明类型  
    - 必须是 `局部变量`
    - `声明`和`初始化`在一起  
    
    var aCar = new Car();  相当于 Car aCar = new Car();
    var c = true/false ? 123:"jiliguagua";
    // 相当于true：int c = 123;
    //      false: String c = "jiliguagua";
    

### 常量  

使用 `final` 关键字  

> `const` 是Java保留字，但并没有使用。定义常量用 `final`  

## 运算符

- 算术运算符  
    - 一元  
    `++` `--` `+` `-`  
    - 二元  
    `*` `/` `%(好像除了Python，都是取余)` `+` `-`  
    - 关于`%`(这里只讨论除数为正(即**n为正数**))  
    
        > 正数 % n，结果为 0 ~  (n-1)  
        > 负数 % n，结果为 -(n-1) ~ 0  
        
        在**保证逻辑**的前提下，  
        使结果总为**正**：`((x % n) + n) % n`  
        使结果总为**负**：`((x % n) - n) % n`  
        - `Math.floorMod(x, n)`实现了上述结果**总为正**的功能
- 关系运算符  
`>` `>=` `<` `<=` `==` `!=`  
- 逻辑运算符  
`!` `&&` `||` `?`  
- 位运算符  
    - 位操作  
    `~` `&` `|` `^`
    - 位移位  
    `<<` `>>` `>>>`  
- 赋值运算符  
`=` `+=` `-=` `*=` `/=` `%=` `...`  

> 复合型赋值运算符隐含**强制类型转换**

## 控制流  

> 块作用域：`{}`包裹的若干语句  
> `Java`不允许在**嵌套**的块作用域定义同名变量  
> `C++`可以，子块的变量会覆盖父块

- 分支  
    - `if else`  
    

if () {
        
} else if () {
        
} else {
        
}


- `switch case`  


switch () {
    case 整型/char/String/enum: ...; break;
    ...
    default: ...
}

    
- 循环  
    - `while/do while`  
    do while 语句后有 `;`  
    `do {...} while ();`  
    - `for`  
    `for ( ; ; ) {}`  
    - 增强for循环  
    用来处理`数组/集合`  
    `for (变量:集合) {}`  
    
    > Java不使用`,`运算符  
    > `,`可在for语句的第1部分、第3部分`分隔`表达式  
    
- `break` `continue` `goto` 以及 `标签语句`  
Java中不使用 `goto`语句  

## 字符串  

- UTF-16  
<img src='../images/UTF-16.png' width='900px'></img>  
- 转义序列  
`\uxxxx`，需引号包裹  

    > Unicode转义序列在解析代码前处理，可改变程序结构  
`"\u0022+\u0022"`并不是字符串`""`，在解析代码前`\u0022`会转换为`"`,因此代码变成了`""+""`,即两个空串相加

概念上，字符串就是 `Unicode字符序列`  

> Java中字符串是**不可变**的  
> 各种字符串存放在公共的存储池中，String变量指向池中相应位置。  
> 如果复制一个字符串变量，两个变量指向相同位置  

- 长度  
`.length()`
- 改  
子串：`String substring(int beginIndex, int endIndex)`  
连接：`String join(String delimiter, String... elements)`  
分隔：`String split(String str)`  
重复：`String repeat(int count)`  
大小写转换：`String toLowerCase()` `String toUpperCase()`  
删除头尾部的字符，小于等于`U+0020`的字符(trim)；空格(strip)：`String trim()` `String strip()`  

- 查   
`int indexOf/lastIndexOf(String str, int fromIndex)`  
`char charAt(int index)`  
`boolean startsWith/endsWith(String str)`  
`boolean equals/equalsIgnoreCase(String other)`  
`int compareTo(String other)`  

> 检测相等一定不要使用`==`，该运算符只能确定两字符串是否在同一位置  
> 虽然同一位置的字符串一定相等，但完全有可能`内容相同的字符串在不同位置上`  

- 增  
    1. s1 `+` s2  
    2. `String.join()`  
    

        String all = String.join(" / ", "S", "M", "L", "XL");  
        // all: "S / M / L / XL"
    
    
3. `repeat();`  


    String repeated = "Java".repeat(3);  
    // repeated: "JavaJavaJava"
    
        
    4. 构建字符串：`StringBuilder`类  
    由大量较短的字符串构建字符串，如按键或文件中的单词，使用`+`效率很低：每次拼接，都会创建一个新的String对象  
    应按以下步骤进行：  
    
        ```  
        StringBuilder builder = new StringBuilder(); // 创建空的字符串构造器
    
        builder.append(x)  
        builder.append(x)  //添加内容x(整型、浮点型、布尔、对象都可以)到末尾  
        builder.insert(index, x) //添加到指定位置  
        builder.delete(start, end)
    
        String allStr = builder.toString() // 取出内容
        ```  

- 空串与`null`  
两者**概念不同**  
`null`是一个特殊的对象  
空串是`""`，长度为0，且没有内容

## 数组  
数组存储**相同类型**的值的序列  
一旦创建，就不能更改数组的**长度**  
- 长度  
`.length`
- 创建数组  


int[] arr = new int[length]; 
method( new int[]{1,3,4,} );//匿名简写
//标准形式与其他形式不可同时使用


数组的数组：  


int[][] arr = new int [n1][n2]
int[][]... arr = new int[][]{{...}, {...}} //匿名简写

    
- 数组的初始化：  
    整型：0  
    浮点型：0.0  
    boolean：false  
    char：空格  
    对象：null  
- `Arrays`类  
打印数组：`Arrays.toString(arr)`  
打印数组的数组：`Arrays.deepToString(arr)`
```
